# ♛ N-Queens Problem Solver

This project demonstrates a Python solution for the N-Queens problem using recursion and backtracking. The program places queens row by row and checks whether each position is safe before moving to the next row.

## 📌 Objectives

- Solve the N-Queens problem using recursive backtracking.
- Place queens on the board without conflicts.
- Use sets to track occupied columns and diagonals.
- Store all valid queen arrangements.
- Practice recursion and logical problem solving in Python.

## 🛠️ Tools & Concepts Used

- Python
- Recursion
- Backtracking
- DFS Logic
- Sets
- Lists
- Constraint Checking
